# Bravo

Phát triển một số module đơn giản!

## Bravo Slider

Code mẫu:

```html
<bravo-slider
  formControlName="dataSlider"
  [options]="optionsSlider"
></bravo-slider>
```

## Bravo Range Time

Code mẫu:

```html
<bravo-range-time
  (timeEvent)="timeEvent($event.minTime, $event.maxTime)"
></bravo-range-time>
```
